## Description

Currently only supporting Windows

## Development

Install dependencies

```bash
npm i
```

Start App in development mode

```bash
npm run dev
```

Bundle App into binaries

```bash
npm run bundle
```

## Package scripts

- `npm run dev` Start App in dev mode
- `npm run bundle` Bundle App into binaries
- `npm run lint` Lint App
